## How to Run the Project
1. Clone the repository:
   ```sh
   git clone https://github.com/Devkumar-web/Twitter-clone.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Twitter-clone
   ```
3. Open `index.html` in a web browser.
